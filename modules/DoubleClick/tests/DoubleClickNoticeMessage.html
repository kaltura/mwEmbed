DoubleClick with a custom notice message

This test page demonstrates displaying a custom notice message during ad playback.
The notice message can be localized and can contain evaluated expressions using curly brackets.
The following Flashvars control the notice message:

  • enableCountDown: Enable / Disable the notice message. If set to true but "countdownText" is not set - a default notice message appears.
  • countdownText: The message text to display. Can contain evaluated expressions.
