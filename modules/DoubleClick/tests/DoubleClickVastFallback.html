DoubleClick with VAST fallback for IE8, IE9.

The DoubleClick plugin is not supported on Internet Explorer 8 and 9.
It is also not supported when implementing the Kaltura player SDK in iOS and Android native applications.
You can provide a VAST fallback for these browsers by adding a VAST plugin configuration.
On IE8, IE9 and mobile native applications, the VAST configuration loads instead of the DoubleClick configuration.
