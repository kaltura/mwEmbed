Conviva Module

Force Mobile HTML5 for testing with desktop html5 browsers

Simply enabling the Conviva plugin via flashVars.
This will result in a run-time error in the Flash player as other required parameters are missing.
The HTML5 module loader will execute up to the point of validating the required parameters, log errors, and continue without loading the actual Conviva library.

This test player behavior is intentional as we are not allowed to expose required parameter values here (legal and NDA implications).
