External Logger

This test page demonstrates the usage of a DIV as an external logger.
The following properties can be set:
  • debug: When set to true, debug messages are logged.
  • debugTarget: The ID of the external logger DOM element.
  • autoScrollDebugTarget: When set to true, the external logger is auto scrolled. Default is false.
  • debugFilter: A string or regular expression used to filter the logged messages.

Logger:


Sample embed code: 
mw.setConfig("debug",true);
mw.setConfig("debugTarget","kdebug");
mw.setConfig("autoScrollDebugTarget",true);
mw.setConfig("debugFilter","Analytics");
kWidget.embed({
	'targetId': 'kaltura_player',
	'wid': '_243342',
	'uiconf_id' : '12905712',
	'entry_id' : '0_uka1msg4',
	'flashvars': {
		'autoPlay': true
	}
});
