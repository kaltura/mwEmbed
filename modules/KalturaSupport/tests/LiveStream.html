Live Stream Example

This demonstrates a simulated live stream with the kaltura player. It broadcasts a sequence kaltura connect 2014 content.

Controlling Live Buffering ( flash player, flashvars )

mediaProxy.expandedLiveBufferTime = default is 0.75 sets the maximum buffer time to be set for a live entry - once the video has started playing and we want to minimize further buffering events.
mediaProxy.initialLiveBufferTime = default 0.75 sets the initial buffer time for a live entry
