Flash embed params


Kwidget.embed supports the param option to pass parmas to the flash embed. 
kWidget.embed( {
		'targetId': 'myVideoTarget',
		'wid': '_243342',
		'uiconf_id' : '12905712',
		'entry_id' : '0_uka1msg4',
		'flashvars':{
			'externalInterfaceDisabled' : false,
		},
		'params' : {
			'wmode' : 'opaque'
		}
	} );
