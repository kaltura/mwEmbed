Capture Thumbnail


Sample embed to enable captureThumbnail. Note you would want to supply a ks 
kWidget.embed( {
	'targetId': 'kaltura_player',
	'wid': '_243342',
	'uiconf_id' : '12905712',
	'entry_id' : '0_uka1msg4',
	'flashvars': {
		'ks': /* your ks with thumbnail capture permission */
		'captureThumbnail':{
			'plugin': true,
			"tooltip": "Capture Thumbnail"
		}
	}
});
