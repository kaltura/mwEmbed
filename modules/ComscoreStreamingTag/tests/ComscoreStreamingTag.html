ComscoreStreamingTag Plugin


Analytics events: ( only html5 )
Mapping Tags
Some of the tags should be derived from video entry metadata yet mapped to a predetermined ComscoreStreamingTag unique ID (such as C3, C4 and C5 described above) and while those don't change often, they are subject to change every once in a while. As a solution the comscorestreamingtag plugin will load an external XML which will be a dictionary the plugin should use to conduct mapping. That loading should be done before content is played back so we don't miss any events. The link to the XML may be configured in the plugin parameters.
Example mapping file: 
 <c3>
    <contentOwner name="RADWANTZOR" id="1894" />
    <contentOwner name="AVGIL" id="8457" />
    <contentOwner name="SHADMI" id="1540" />
    <contentOwner name="KRIMOV" id="9841" />
    <contentOwner name="YEFET" id="9813" />
  </c3>
